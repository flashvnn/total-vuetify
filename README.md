# total-vuetify

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run client:dev
```

Then run debug command for total.js

```
node debug.js
```

### Compiles and minifies for production
```
npm run build
node release.js
```

### Lints and fixes files
```
npm run lint
```
